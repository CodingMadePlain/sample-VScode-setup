# ✨ Sample VS Code Setup For Agentic Coding
**By: Dele Oke**

What is Agentic Coding is where AI models autonomously  plan, implement, and debug code based on clear instructions and structured requirements.

---

## Prerequisites 
- VS Code installed
- [Git 2.23+](https://git-scm.com/)
- [GitHub account](https://github.com/)
- GitHub Copilot subscription (or trial)

## Overview of Github Copilot in VS Code

Github Copilot is the *AI Assitant* and the *LLM's* are supplied by:
- OpenAI (GPT family e.g GPT-4)
- Anthropic (Claude family e.g Claude 2)
- Google (Gemini family)
- And others



## Sample Setup
```text
.
├── .github/
│   ├── copilot-instructions.md       # References AGENTS.md
│   ├── instructions/                  # Coding standards (HTML, CSS, JS)
│   ├── prompts/                       # Reusable AI prompts
│   └── skills/                        # Agent capabilities
├── AGENTS.md                          # Agent contextual instructions
├── .gemini/                           # Gemini-specific configs
├── .gitignore
├── workarea/                          # AI-generated code (sandbox)
├── projects/                          # Large multi-folder projects
├── notes/                             # Tutorial and student notes
├── large_assets/                      # Non-tracked large files
├── prds/                              # Product Requirement Documents
└── README.md
```
